1.安装node.js
2.安装react native command line
    安装npm install -g react-native-cli
    react-native --help
    初始化一个项目:react-native init FirstApp
    运行ios项目:
        react-native run-ios
        -or-
         Open firstApp\ios\firstApp.xcodeproj in Xcode or run "xed -b ios"
    运行android项目:
         Have an Android emulator running (quickest way to get started), or a device connected.
         cd E:\React\firstApp && react-native run-android
3.android studio(windows)/xcode(mac)

执行react-native start 启动包服务器
http://localhost:8081/index.android.bundle?platform=android  检查是否成功
执行命令：nox_adb.exe connect 127.0.0.1:62001，连接模拟器
run app 运行项目

生成assets文件夹内容:
react-native bundle --platform android --dev false --entry-file App.js --bundle-output android/app/src/main/assets/index.android.bundle --assets-dest android/app/src/main/res
